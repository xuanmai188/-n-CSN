Test Admin URL Login System

Hướng dẫn test:

Hệ thống đã được cập nhật để xóa link "Admin Panel" từ footer và thay thế bằng đăng nhập qua URL parameter.

Cách đăng nhập Admin mới

Thêm ?a vào cuối URL:

  • index.html?a
  • products.html?a
  • contact-us.html?a
  • about-us.html?a
Tài khoản admin: admin / admin123
Đã xóa khỏi Footer

Link "Admin Panel" đã được xóa khỏi footer của tất cả các trang:

  • index.html
  • products.html
  • about-us.html
  • contact-us.html
  • detail.html
  • Và tất cả các trang khác...
Bảo mật cao hơn - không hiển thị công khai
Cách hoạt động
  1. Khi truy cập URL có parameter ?a, JavaScript sẽ tự động hiển thị modal đăng nhập admin
  2. Modal sử dụng function showAdminLogin() có sẵn trong auth-cart.js
  3. Sau khi đăng nhập thành công, sẽ chuyển hướng đến admin.html
  4. Link admin không còn hiển thị công khai trong footer
